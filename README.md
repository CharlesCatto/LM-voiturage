# LM Covoiturage

## 🚗 Objectif
Faciliter le covoiturage entre employés de Leroy Merlin Dourges 1.

## 🗺️ [Voir la carte en ligne](https://charlescatto.github.io/LM-voiturage/)

## Fonctionnalités
- Carte interactive avec clustering
- Recherche par nom/prénom ou localité
- Popups avec photos et infos employés
- Design mobile-first

## Données
- 12 employés fictifs
- 3 équipes (matin, après-midi, nuit)
- Secteurs : Béthune, Arras, Douai/Lièvin
